This is code for a robot that I am working on for my Embedded systems class here at MSU. I have chosen to leave this repository public so that any recruiters may look and see some of the code I have done.

The brain of this robot is a rasberry pi 5 and the muscles are servos. This robot runs mainly on python and a little bit of html and javascript.

Right now the robot has a functioning flask webpage that I can use to control its movements and speach.

what I am currently working on as i begin to finsh the second project is a dialoge engine that can understand some text inputs and can change states and invoke actions for the robot.

As I get further into building the robot I will add commits to show progress.

This project was made with the help of AI like chatgpt as it is encouraged by the professor.

I hope to add some features to it on my own time in the future.
